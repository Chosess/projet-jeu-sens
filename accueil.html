RETRO PUNK

Règle du jeu

Règles du jeu :

Au sein de ce jeu, ton objectif est d'éviter les obstacles, collecter des checkpoints et survivre.


Contrôles :

Utilise les touches directionnelles gauche, droite pour déplacer le joueur dans le jeu.


Le joueur :

Au début du jeu, le joueur dispose de 3 vies.

Chaque fois que le joueur entre en collision avec un obstacle, il perd une vie.

Si le joueur perd ses 3 vies, la partie est terminée et il a perdu.


Les checkpoints :

Les checkpoints sont des objets dans le jeu que le joueur doit collecter.

Chaque fois que le joueur collecte un checkpoint, il gagne 1 point.

L'objectif est de collecter 10 checkpoints pour remporter la partie.


Les obstacles :

Les obstacles sont des objets mobiles ou statiques dans le jeu qui peuvent causer des collisions.

Si le joueur entre en collision avec un obstacle, il perd une vie.


Collision avec les bords du jeu :

Si le joueur entre en collision avec les bords du jeu, cela est considéré comme une collision avec un obstacle, et il perd une vie.


Affichage des informations :

Affiche le score actuel du joueur (nombre de checkpoints collectés).

Affiche le nombre de vies restantes du joueur.

Affiche un message lorsque le joueur gagne en collectant 10 checkpoints.

Affiche un message lorsque le joueur perd en perdant ses 3 vies.


Fin du jeu :

Si le joueur collecte 10 checkpoints, affiche un message de victoire et termine la partie.

Si le joueur perd ses 3 vies, affiche un message de défaite et termine la partie.
